# Dominio de Logística

## Propósito y alcance
Coordina el fulfillment, almacenamiento, transporte y devoluciones físicas de los pedidos. Integra centros de distribución, transporte propio y terceros para garantizar entregas y recolecciones oportunas.

## Equipo y usuarios
- **Equipo operativo:** planificadores de demanda, líderes de centros de distribución y especialistas en última milla.
- **Usuarios internos:** operadores de almacén, coordinadores de transporte, analistas de seguimiento.
- **Usuarios externos:** transportistas aliados, couriers, clientes que gestionan retiros y personal de tiendas.

## Responsabilidades en el flujo del pedido
- Recibir órdenes confirmadas y generar tareas de picking, packing y despacho.
- Gestionar la trazabilidad de envíos y devoluciones, informando estados clave.
- Coordinar con atención al cliente reprogramaciones, incidencias y recolecciones.
- Validar la recepción de devoluciones para disparar procesos contables.

## Historias de usuario
1. Como **coordinador de transporte** quiero asignar rutas de entrega basadas en SLA y capacidad para optimizar tiempos y costos.
2. Como **operador de almacén** quiero acceder a las tareas de picking priorizadas para preparar pedidos urgentes primero.
3. Como **transportista aliado** quiero confirmar la entrega desde mi dispositivo móvil para cerrar la orden en tiempo real.
4. Como **analista de devoluciones** quiero registrar el estado físico del producto devuelto para informar a contabilidad y ventas digitales.
