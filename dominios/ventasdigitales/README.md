# Dominio de Ventas Digitales

## Propósito y alcance
Gestiona la experiencia de compra digital: navegación, carrito, pricing, promociones, checkout y confirmación de pedidos. Orquesta la creación del pedido y publica eventos para el resto de dominios.

## Equipo y usuarios
- **Equipo de producto digital:** product managers, UX y desarrolladores web/app.
- **Usuarios internos:** agentes de soporte comercial, analistas de pricing y catálogo.
- **Usuarios externos:** clientes finales y comercios aliados que venden a través del marketplace.

## Responsabilidades en el flujo del pedido
- Capturar la intención de compra y validar inventario y promociones.
- Procesar pagos o integrar con pasarelas externas.
- Emitir eventos de orden confirmada para logística y contabilidad.
- Actualizar al cliente sobre el estado del pedido y reflejar devoluciones o notas de crédito.

## Historias de usuario
1. Como **cliente digital** quiero confirmar mi pedido con múltiples métodos de pago para completar la compra sin fricciones.
2. Como **agente de soporte comercial** quiero duplicar un pedido en nombre del cliente para asistirlo si el checkout falla.
3. Como **analista de catálogo** quiero actualizar precios y disponibilidad en tiempo real para evitar ventas de productos sin stock.
4. Como **seller del marketplace** quiero ver el resumen de mis pedidos entrantes para preparar mis despachos a tiempo.
